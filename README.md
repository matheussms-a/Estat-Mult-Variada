Este repositório é referente ao trabalho prático final da disciplina de Estatística Multivariada 2 do Departamento de Estatística da Universidade Federal de São Carlos (UFSCar). 
Sob orientação do Prof. Dr. Pedro Ferreira Filho, o projeto explora técnicas avançadas de análise de dados aplicadas a problemas reais.

O estudo principal utiliza a Análise Fatorial de Correspondência Múltipla (AFCM) para compreender a estrutura de preferência de estudantes em relação a 10 marcas de cerveja.
A análise foi construída através da categorização de notas atribuídas pelos consumidores, permitindo a visualização de relações complexas em planos fatoriais e a interpretação de tabelas de Burt percentuais para identificar perfis de consumo.

O relatório foi produzido utilizando LaTeX para garantir o rigor técnico na apresentação de tabelas e gráficos, enquanto o processamento estatístico foi realizado em linguagem R, utilizando bibliotecas como FactoMineR e factoextra.
Os projetos contam com a autoria de Matheus Santello.
